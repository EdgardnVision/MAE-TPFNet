**MAE-TPFNet: Pole-Fall State Recognition in Distribution Network Construction Surveillance Videos**

MAE-TPFNet is a clip-level video recognition framework for pole-state recognition in distribution network construction scenarios. The task is formulated as a three-class risk-state recognition problem, including **Normal**, **Tilt**, and **Falling**, corresponding to safe, warning, and alarm states, respectively.
